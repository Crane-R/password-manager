# 密码管理器 Password Manager v8.0-beta 功能说明

| 锁界面 | 功能    | 说明                                                                          |
|-----|-------|-----------------------------------------------------------------------------|
| 1   | 新增场景  | 单击后进入新增场景模式，输入一个新的密钥可直接进入一个全新的场景                                            |
| 2   | 进入场景  | 单击后进入登录场景模式，输入已存在密钥进入场景                                                     |
| 3   | 多语言   | 单击功能介绍按钮右边的倒三角按钮弹出多语言列表，单击任意一项可重启进入选择的语言界面（除中文外的语言可能兼容性不高，比如字太长等，并且可能有文本缺失） |
| 4   | 数据库模式 | 单击后进入数据库模式，实质上是将数据存储载体切换到了数据库，数据库可能需要进行配置才能使用                               |
| 5   | 文件模式  | 单击后进入文件模式，数据存储载体切换为excel文件，该模式不需要进行任何配置                                     |
| 6   | 远端数据库 | 单击后将数据库配置切换为远端服务器（程序带有远端数据库配置和本地数据库配置）                                      |
| 7   | 本地数据库 | 单击后将数据库配置切换为本地                                                              |

| 主界面 | 功能     | 说明                                         |
|-----|--------|--------------------------------------------|
| 8   | 活性时间   | 指一定时间内未进行操作则返回至锁界面的安全手段，时间可在配置修改，该功能可在配置关闭 |
| 9   | 锁定     | 锁定活性时间                                     |
| 10  | 解锁     | 解锁并刷新活性时间                                  |
| 11  | 锁 \| 搜索 | 搜索账户并使其处于加密状态                              |
| 12  | 解码     | 将搜索结果全部解码                                  |
| 13  | 新增账户   | 关闭主界面打开新增界面                                |
| 14  | 清空当前   | 清空搜索结果和消息输出框                               |
| 15  | 双击模式   | 单击回车搜索，双击回车搜索并解码，双击判断间隔为200毫秒              |
| 16  | 实时模式   | 根据文本框值的增减、改变进行实时搜索                         |
| 17  | PM3.0  | 该功能为v3.0准备，如导入非v3.0的数据，可能会导致数据错误等意外情况。     |
| 18  | PM4.2  | 该功能为v4.2准备，如导入非v4.2的数据，可能会导致数据错误等意外情况。     |
