Password Manager v8.0-beta 功能说明 Beta

序号软件特点介绍
1存储模式策略文件模式：存储在用户本地的电脑，位于软件安装目录下的resources\light_weight_data
数据库模式：可配置本地数据库和远端服务器数据库
2加密策略加密算法具体见软件说明文档
序号关于功能描述/解释
3数据备份问题因为在文件模式下所有数据都是存储在用户电脑本地，所以请用户自行做好数据的备份（如定期使用导出功能导出数据）
4活性时间活性时间到期后会回退到锁界面，在时间即将到期（10%）时会变为红色
5数据库模式基于该软件的数据量、安全性、便捷性等方面的综合考虑，数据库模式在4.x后的版本已经不再维护
但依旧可以经过配置文件的修改使用，具体的部署方案（如数据库表的字段是固定的）需要询问开发者
