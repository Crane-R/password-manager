Password Manager v8.0-beta 功能说明 Beta

锁界面功能说明
1新增场景单击后进入新增场景模式，输入一个新的密钥可直接进入一个全新的场景
2进入场景单击后进入登录场景模式，输入已存在密钥进入场景
3多语言单击功能介绍按钮右边的倒三角按钮弹出多语言列表，单击任意一项可重启进入选择的语言界面（除中文外的语言可能兼容性不高，比如字太长等，并且可能有文本缺失）
4数据库模式单击后进入数据库模式，实质上是将数据存储载体切换到了数据库，数据库可能需要进行配置才能使用
5文件模式单击后进入文件模式，数据存储载体切换为excel文件，该模式不需要进行任何配置
6远端数据库单击后将数据库配置切换为远端服务器（程序带有远端数据库配置和本地数据库配置）
7本地数据库单击后将数据库配置切换为本地
主界面功能说明
8活性时间指一定时间内未进行操作则返回至锁界面的安全手段，时间可在配置修改，该功能可在配置关闭
9锁定锁定活性时间
10解锁解锁并刷新活性时间
11锁 | 搜索搜索账户并使其处于加密状态
12解码将搜索结果全部解码
13新增账户关闭主界面打开新增界面
14清空当前清空搜索结果和消息输出框
15双击模式单击回车搜索，双击回车搜索并解码，双击判断间隔为200毫秒
16实时模式根据文本框值的增减、改变进行实时搜索
17PM3.0该功能为v3.0准备，如导入非v3.0的数据，可能会导致数据错误等意外情况。
18PM4.2该功能为v4.2准备，如导入非v4.2的数据，可能会导致数据错误等意外情况。
